Get Started

Up and running in 2 minutes

PowerShell 
# Clone & enter
git clone https://github.com/AmrDab/clawd-cursor.git
cd clawd-cursor

# One-command setup (installs TightVNC + builds everything)
powershell -ExecutionPolicy Bypass -File setup.ps1

# Start the agent
npm start -- --vnc-password yourpass
Setup script handles: Node.js check · TightVNC download & silent install · npm install · TypeScript build · .env creation
Requires: Node.js 20+ · PowerShell · AI API key (optional — works offline for common tasks)

Architecture

Two paths, one goal

Your provider determines the execution path. Anthropic gets native Computer Use. Everyone else gets the fast router.

A

Computer Use (Anthropic)

Full task → Claude with native desktop tools. Screenshots + planning + execution in an autonomous loop. Handles any complexity.

🧠 Autonomous · Reliable
B

Action Router (OpenAI/Offline)

Decompose → UI Automation tree → find elements by name → execute. Zero LLM for common patterns like opening apps and typing.

⚡ Fast · $0 for simple tasks
→

VNC Execution

Both paths converge on the same VNC client: mouse clicks, keyboard input, drag operations, screenshots — all via standard VNC protocol.

🔌 Any VNC server
Features

Built for speed, safety, and reliability

🔍

UI Automation Tree

Queries native accessibility APIs on Windows, macOS, and Linux to find buttons, menus, and inputs by name — not pixel coordinates.

⚡

Zero-LLM Fast Path

PATH B: Common tasks execute in milliseconds with zero AI calls. Open apps, type text, click elements — instantly via UI Automation.

🧠

Native Computer Use

PATH A: Claude gets native desktop tools — screenshot, click, type, scroll, drag. Plans and executes complex workflows autonomously.

📸

Adaptive Screenshots

Screenshots scaled to 1280×720 for the API. Adaptive delays per action type — 1s for app launch, 100ms for typing.

🛡️

Safety Tiers

Three-tier system: auto for safe actions, preview for typing, confirm for anything destructive.

🔌

VNC Protocol

Connects as a standard VNC client. Watch the AI work in real-time from your own viewer.

🟢 Auto

Open apps, navigate, read screen, switch windows. Executes immediately — no confirmation needed.

🟡 Preview

Type text, fill forms. Logged before executing so you can see exactly what's happening.

🔴 Confirm

Send messages, delete files, purchases. Pauses and waits for your explicit approval.
